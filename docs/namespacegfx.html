enum gfx::vlTransferType

Specifies the type of synchronization strategy for buffer or image usage.

Enumerator
VL_TRANSITION_AUTO 

Adds a barrier only if a transition is required.

VL_TRANSITION_TRANSITION 

Forces a transition regardless of the previous state.

VL_TRANSITION_NONE 

Suppresses any transition, assumes user knows state is valid.
