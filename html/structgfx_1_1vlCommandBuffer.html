virtual void gfx::vlCommandBuffer::flushPendingTransitions ( )
pure virtual

Forces emission of all deferred transitions.

Should be called before any draw/dispatch to ensure resources are in the correct state. This flushes queued transitions into appropriate vkCmdPipelineBarrier calls.

vlTransferType transferType = VL_TRANSITION_AUTO )
pure virtual

Declares usage of a buffer with optional deferred access synchronization.

Like useImage, it determines whether a pipeline barrier is required and defers the decision to allow batching transitions more efficiently.

Parameters
bufferTarget buffer resource.
accessAccess rights required.
stagePipeline stage of the access.
transferTypeDetermines if/when the transition is emitted.

pure virtual

Declares usage of an image with optional deferred layout transition.

Automatically queues an image transition based on access and layout requirements. The actual transition may be deferred and grouped with others before execution.

Parameters
imageTarget image resource.
desiredLayoutLayout the image should be in when used.
accessAccess rights required.
stagePipeline stage of the access.
transferTypeDetermines if/when the transition is emitted.
aspectAspect mask to be transitioned.
levelCountMipmap levels affected.
layerCountArray layers affected.
